## Overview

This project implements a full-stack real-time chat system designed to mirror
modern messaging platforms.

It explores how real-time communication works at scale, including persistent storage,
event-driven updates, and efficient state synchronization between clients.

Core concepts include:

- WebSocket-based communication for low-latency messaging.
- Database design for relational chat systems.
- Caching and pub/sub mechanisms for scalability.
- Separation of concerns in backend architecture.
